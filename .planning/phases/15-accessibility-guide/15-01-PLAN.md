**Established patterns from v1.1:**
- crypto.randomUUID() for unique ARIA IDs
- FocusTrap component for modal focus management
- Roving tabindex for arrow key navigation
- aria-describedby for error messages
- role/aria-* attributes following WAI-ARIA patterns
- focus:ring-2 focus:ring-blue-500 for focus indicators
- PrefersReducedMotion() for motion preferences
- axe-core + Playwright for automated testing

   - Alertdialog pattern (ConfirmDialog) - role="alertdialog" for destructive confirmations
   - Combobox/listbox pattern (CommandPalette, Combobox) - roles, activedescendant, expanded state
   - Tablist pattern (Tabs) - role=tablist/tab/tabpanel, aria-selected, roving tabindex

   - Live regions (Alert, Toast, Progress, Spinner) - role=alert/status, aria-live
   - Form controls - label association via htmlFor/id, aria-invalid, aria-describedby for errors
   - Navigation - role=navigation, aria-current for active items

3. **Unique ID Generation** - crypto.randomUUID() pattern for aria-labelledby/describedby links
